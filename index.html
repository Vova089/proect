Как создать свой сайт?


Многие задаются вопросом "как создают сайт?"и не знают на каком языке программирования создают.

HTML переводиться, как Hyper Text Markup Language и используется для создания сайта.

С чего начать?

Стоит начать с изучения тегов и базовых стилей

Тегов много и каждый из них делает свою работу. Например: тег < a > нужен для создания сылок, а тег < img > нужен для создания картинок.

После следует закрепить материал. Например создать сайт без домена и хостинга.

Как опубликовать свой сайт?

После практики можно попробовать сделать свой сайт, но как?

  1. Сделайте макет сайта.
  2. Напишите код сайта пользовавшись макетом.
  3. Проверьте сайт.
  4. Купите домен и хостинг.
  5. Сайт опубликован!

Также можно сделать логотип для сайта в програмке Figma, украсить сайт, добавить движений, картинки и сылки
